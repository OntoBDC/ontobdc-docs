# OntoBDC

**Ontology-Based Data Capabilities**

OntoBDC es un runtime orientado a capabilities para la ejecución de operaciones de datos conscientes de ontología sobre datasets portables. Cierra la brecha entre el almacenamiento de datos estático y la ejecución semántica dinámica.

## Características Principales

- **Portabilidad Semántica**: Mueva sus proyectos (Data Package + RO-Crate) a cualquier lugar; el sistema los entiende al instante.
- **Capability Runtime**: Descubra y ejecute operaciones modulares basadas en el contexto de los datos.
- **Columna Vertebral de Eventos**: Rastree cada cambio en el ciclo de vida de sus datos con una arquitectura robusta de event sourcing.
