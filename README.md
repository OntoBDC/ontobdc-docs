# OntoBDC Stack

Ontology-Based Data Capabilities stack.

## Documentation

The official documentation is available in the `page/` directory and is built using MkDocs.

To view documentation locally:
```bash
cd page
pip install -r requirements.txt
./build.sh
mkdocs serve
```

## Repository Structure

- `core`: Core library distribution.
- `wip`: Development workspace for new capabilities.
- `page`: Documentation site source.
- `config`: Configuration files.
